#MANUAL TÉCNICO DE LA APLICACIÓN
En este proyecto encontrarán un MENU PRINCIPAL que ejecutará otros menús dependiendo de la elección del usuario.

QUE INCLUYE
Cada menú ejecutará una función independiente y almacenará los datos en un documento json 
Cada docuento tiene modulación lo que significa que se estarán importando las funcionalidades de un lado a otro.
Hay validaciones dentro de estas funciones que permiten por ejemplo verificar la disponibilidad antes de asignar instructores o clientes, deberán ser registrados primero para garantizar la privacidad y eficiencia de las citas. 


El proyecto no incluye almacenamiento de errores de usuario en otros archivos. Tampoco deja borrar datos.
